JUNGRAPHT

Java Universal Network/Graph Framework Ported to JGraphT

View project on GitHub

JUNGRAPT is a port of the Java Universal Network/Graph Visualization Framework that uses JGraphT graph models and algorithms. Junggrapht (pronounced 'June Graph Tea') includes many performance enhancements that are not present in the original JUNG visualization code. Force Directed Layout algorithms are enhanced with a Barnes Hut Quad Tree. Visualizations are enhanced with an R*Tree for nodes and edges. Additionally, there is a swapping rendering stage that greatly improves rendering performance while a graph visualization is being transformed by mouse gestures, or is zoomed out to a scale that is too small for details.
